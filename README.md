# Week4 项目

本目录包含第4周的学习内容。

## 目录结构

- `week4/` — 第4周作业与练习
  - `0608.ipynb` — 成绩可视化分析 + IoU 计算 + 特征相似度检索
  - `0608.py` — Python 脚本版本
  - `0609.ipynb` — 后续练习
  - `figures/` — 图片资源
  - `file/` — 数据文件（特征库）
    - `feat.txt` — 特征库数据
    - `target_feat.txt` — 目标特征数据
  - `test/` — 测试脚本

## 内容概要

1. **成绩可视化分析** — 使用 Matplotlib 绘制柱状图、直方图、散点图、折线图
2. **IoU 计算** — 目标检测中候选框与真实框的交并比计算
3. **特征相似度检索** — 基于余弦相似度查找最相似的特征
